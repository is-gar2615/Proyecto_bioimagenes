# Proyecto_bioimagenes
Proyecto visualizador DICOM 3D para la clase de procesamiento de bioimágenes 2025B

**Los archivos DICOM fueron obtenidos de esta pagina:** https://3dicomviewer.com/dicom-library/
Asegurarse de cambiar la dirección de los archivos en el programa.

Los archivos 3DDICOM.py Y xxxxx Son de prueba.


**Archivos finales de uso son:** 3DITK.py y 3DSegmentanción.py
